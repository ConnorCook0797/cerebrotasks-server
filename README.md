# CerebroTasks Server

Small starter backend for Railway.

## Files
- `index.js` - Express server
- `package.json` - dependencies
- `.env.example` - example environment variables

## Local run
1. `npm install`
2. set `DATABASE_URL`
3. `npm start`

## Routes
- `GET /`
- `GET /health`
- `GET /setup`
- `POST /obsidian/task-change`
